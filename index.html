使用的软件

Geektool. 它的主要用途就是把终端输出的结果投影到桌面上，然后可以设置刷新频率什么的，让你的桌面看上去比较碉堡。

它是免费软件。你也可以使用 Homebrew 获得它。

brew cask install geektool
