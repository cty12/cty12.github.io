前言

用到的工具

  • 首先你得有一只 Mac 娘，无论是 MacBook 还是 iMac 都无所谓。我用的是 MacBook Pro late 2013 。

  • 其次她得装有 python ，我用的是 2.7.5 。不过请你不用担心 Mac 娘通常情况下都会自带这个版本（也许是更老版本的）的 python 。
