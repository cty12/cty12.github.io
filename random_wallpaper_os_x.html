OS X 自动更改壁纸

Date 六 05 七月 2014 By cty Category tech. Tags Mac OS X python

前言

本人和很多同学一样，有选择恐惧症。比如在换壁纸的时候，我经常犹豫不决是换哪个壁纸比较好，所以我就写了一个简单的 python 脚本来“替”我选择。

用到的工具

  • 首先你得有一只 Mac 娘，无论是 MacBook 还是 iMac 都无所谓。当然如果你用 Linux 的话原理也差不多（不过你都用 Linux 咯还要看这篇文干什么！）。我用的是 MacBook Pro late 2013 。

  • 其次她得装有 python ，我用的是 2.7.5 。不过请你不用担心 Mac 娘通常情况下都会自带这个版本（也许是更老版本的）的 python 。

  • 以后我们也许会用到 Geektool ，那是一个桌面增强软件，其效果跟 Linux 上的一个我忘记叫什么名字的差不多。你可以免费获得它，我的方法是通过 Homebrew ，那是一个很好用的包管理器。

实现

我们使用 python 实现，当然你也可以使用其它的方式本质上没什么区别。换壁纸是用的一段 AppleScript 脚本。

我的壁纸通常是存在 ~/Pictures/wallpapers 这个路径下。如果你的壁纸不在这里，请替换 file_dir 。

#! /usr/bin/env python
# -*- coding: utf-8 -*-
# 将壁纸随机设置

import os, getpass, random, subprocess

file_dir = '/Users/' + getpass.getuser() + '/Pictures/wallpapers'
# 如果想使用系统 in stock 的壁纸请使用这个路径：
# file_dir = '/Library/Desktop Pictures'

file = []

for r in os.walk(file_dir):
    for i in r[2]:
        if i[0] != '.':
            file.append(i)

# debug info
cnt = 0
for i in file:
    print str(cnt) + '\t -- \t' + i
    cnt = cnt + 1

file_name = file_dir + '/' + file[random.randint(0, len(file) - 1)]

SCRIPT = """/usr/bin/osascript<<END
tell application "Finder"
set desktop picture to POSIX file "%s"
end tell
END"""

def set_wallpaper(file_name):
    subprocess.Popen(SCRIPT%file_name, shell=True)

set_wallpaper(file_name)

好了，现在我们新建一个文本文件把它存进去，修改权限变成一个Unix可执行文件：

touch rand_wallpaper
nano rand_wallpaper
sudo chmod u+x rand_wallpaper

实现效果

大功告成！每次你运行这个程序就可以得到一张随机的壁纸啦。
